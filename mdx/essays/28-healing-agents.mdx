I started writing code in 1999.
I dabbled with programming before then, but I'd date the beginning of my identity as a programmer to my first college programming course: CS50^[Before it was big and ballooned.].
From then on I wrote code most days.
Based on my GitHub history, since arriving at Illinois in 2017, I committed to projects 4 out of 5 days^[I probably wrote code some days and didn't commit, but also not all of these commits are source code. 80% is probably not far off.].

I stopped writing code in 2026.
I began using Claude Code in July 2025.
Initially I would review edits and sometimes make changes by hand.
Eventually the agents got better and I got better at collaborating with them.
I turned on YOLO mode and stopped reading diffs.
At some point I realized that I couldn't remember opening an IDE and removed them from my launcher.
I started [small fun projects](https://playlist-party.com/) done [entirely through conversation](https://playlist-party.com/create) and then [larger more serious projects](https://www.usingandunderstanding.ai/) done [the same way](https://www.usingandunderstanding.ai/create).

I'm much happier now.
It's not because I'm not working.
I'm working more than ever.
My [challenge now is stopping](https://steve-yegge.medium.com/the-ai-vampire-eda6e4f07163).

Initially I thought I was happier because I was making more progress and doing a better job at the things I care about: creating new courses, improving existing ones, building new educational technology.
But as I've become accustomed to what I can accomplish with AI, I've realized something else.
I'm happier because I'm not writing code.

[Did](-) I ever love to code?
In English, the word love has become so overloaded with meaning as to be problematically vague.
Greek invites more precision.
I loved creating things (ποίησις).
I loved being good at it (Φιλοτιμία).
I loved that it made me feel special (τιμή).

But did I ever love just doing it (ἔρως)?
The fact that I abandoned it so quickly probably says something.
I know developers who are choosing to continue to participate in coding even while working with AI: by approving edits, writing their own tests, or debugging by hand.
I'm not.
I'm thrilled by the opportunity to work with them through pure conversation, and
excited by their growing abilities to complete development tasks.
I want this to work.
I don't want to write code anymore.
