2021-06-03 : Recipe Programming

Recipe programming comprises working of a set of instructions to produce a program that achieves some desired outcome. The language, environment, and task most strongly associated with recipe programming is Python in a Jupyter Notebook to perform data analysis and visualization. While recipe programming is powerful and useful, it should not be confused with scratch programming, particularly when teaching CS1.
